# Soccer Coach Board

A deployable static HTML, CSS, and JavaScript soccer coaching board.

## Files

- `index.html` — page structure
- `styles.css` — visual design and responsive layout
- `script.js` — interaction logic

## Features

- Drag home and away players
- Switch between 4-3-3, 4-4-2, 4-2-3-1, and 3-5-2
- Draw arrows and lines on the pitch
- Toggle home and away teams
- Add a coach note
- Flip the board
- Clear drawings or reset the board
- Export board as a PNG

## Deployment

Upload the three files to the same directory on your website.

For GitHub Pages, place them in your repository root or `/docs` folder, then enable Pages in your repository settings.
