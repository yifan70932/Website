Quant Watchlist

Daily-updating quantitative cross-sectional analysis of a watchlist of US large-cap equities and ETFs. Built as a learning exercise in modern multi-factor screening practices.

View latest report →

What's in the report

  • Cross-sectional screener — sortable table of risk and return metrics across the watchlist
  • QVM+L factor scores — Quality, Value, Momentum, Low-Vol percentile rankings as practiced by MSCI, S&P, and AQR
  • Cross-sectional charts — normalized cumulative returns, risk vs. return scatter with Sharpe iso-lines, drawdown comparison, hierarchically-clustered correlation heatmap
  • Mean-variance optimization — long-only efficient frontier with GMV, tangency, equal-weight, and risk-parity reference portfolios
  • Strategy backtest — six rule-based strategies (buy & hold, equal weight, risk parity, SMA crossover, 12-1 momentum, mean reversion) with realistic transaction costs
  • Per-ticker technical drilldowns with SMA, Bollinger bands, RSI, and MACD
  • Curated academic literature linking each method to its primary source

How it's built

A Python analyzer runs daily on GitHub Actions, generates a static HTML report, and commits it to this repo. GitHub Pages serves it directly. No server, no live computation per visitor — just a static file that gets refreshed once a day.

Source code lives in _analyzer/ in the repo root. The workflow definition is at .github/workflows/build-report.yml.

Disclaimer: Educational and demonstrative use only. Past performance does not predict future returns. Single-period rankings are noisy estimates subject to multiple-testing concerns (Harvey, Liu & Zhu 2016). Not investment advice.
